# 道家核心论著四层级分类（司衡底层的道体系参考基准）

> 道家核心论著按本源-黄老-玄学-道教承袭四大层级分类，作为司衡底层道体系建模的阅读与引用优先顺序。

## 一、道家本源双核心（必读根基，万道之源）

### 1. 《道德经》（《老子》）

**作者**：老子（李耳）
**定位**：道家开山之作，全书仅五千言，核心为道为本源、无为、有无、自然、不争。整个道家思想总纲，所有道家典籍的理论根基。

### 2. 《庄子》（《南华经》）

**作者**：庄子及其后学
**定位**：先秦道家巅峰，分内篇7（庄子自作，核心）、外篇、杂篇。核心为逍遥齐物、心斋坐忘、万物齐一。拓展精神自由、修身境界，是道家心性修炼代表。

## 二、黄老道家典籍（治国+内修合一，西汉主流道家）

融合老子道论与刑名治国、养生方术，侧重"内圣外王"。

### 1. 《列子》（《冲虚真经》）

**作者**：列御寇
**定位**：以寓言讲虚空、顺性、生死，介于老庄之间，侧重虚无顺自然。

### 2. 《黄帝四经》（马王堆帛书）

**内容**：《经法》《十大经》《称》《道原》
**定位**：失传两千年，出土后重现黄老本源。以道统御法治，汉代治国核心典籍。

### 3. 《淮南子》（《淮南鸿烈》）

**编者**：刘安
**定位**：集西汉黄老大成，融合道、阴阳、儒、法，讲解宇宙生成、修身、治国、天文地理。

### 4. 《鹖冠子》

**定位**：黄老政论，论道、兵、法度，侧重道家治国思想。

## 三、魏晋玄学核心（老庄义理思辨，道家哲学巅峰）

侧重义理辨析、名士心性，融合儒道。

| 典籍 | 作者 | 定位 |
| ---- | ---- | ---- |
| 《周易注》 | 王弼 | 以老解易，玄学奠基，阐发贵无论（道即无） |
| 《庄子注》 | 郭象 | 独化论，魏晋解庄最高权威注本 |
| 《列子注》 | 张湛 | 列子义理注疏 |
| 《世说新语》 | 刘义庆 | 记录玄谈，非理论原著（辅助） |

## 四、道教承袭道家思想的重要典籍

### 早期修身类

- **《太平经》**：最早道教经典，承道家养生、天人合一
- **《抱朴子》**（葛洪）：内篇讲道家炼丹养生、守道；外篇治国
- **《黄庭经》（内景/外景）**：道家内炼、存思、精气神核心典籍

### 唐、宋重道经书（官方册封道经）

| 典籍 | 定位 | 特征 |
| ---- | ---- | ---- |
| 《通玄真经》（文子） | 文子传承老子，黄老早期典籍 | 文子著 |
| 《阴符经》 | 天人、盗机、阴阳大道 | 短小精悍，后世修道必读 |
| 《清静经》 | "大道无形，生育天地" | 极简修行纲领，流传最广入门经文 |
| 《心印经》 | 内丹修炼核心短篇 | 极简 |
| 《悟真篇》 | 张伯端著，内丹修炼第一典籍 | 与《参同契》并称双璧 |
| 《周易参同契》 | 魏伯阳著，万古丹经王 | 以易、道论内外丹 |

## 五、精简优先级（司衡融合道家架构的阅读顺序）

| 优先级 | 分类 | 典籍 | 理由 |
| ------ | ---- | ---- | ---- |
| 1 | 底层理论根基 | 《道德经》《庄子》 | 必看，无可替代 |
| 2 | 宇宙与治国体系 | 《黄帝四经》《淮南子》 | 道论落地为治理框架 |
| 3 | 心性与修身入门 | 《清静经》《阴符经》《列子》 | 操作层面的心性指导 |
| 4 | 思辨拓展 | 王弼《老子注》、郭象《庄子注》 | 义理深化的参考层 |
| 5 | 修炼、精气神体系 | 《周易参同契》《悟真篇》《黄庭经》 | 丹道实操体系 |

## 关键区分：道家 vs 道教

- **道家（先秦哲学流派）**：核心仅《老子》《庄子》《列子》《黄帝四经》《淮南子》
- **道教（宗教）**：依托道家哲学衍生，多出丹道、养生、祭祀典籍
