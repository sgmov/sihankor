## 任务

你是一个哲学辩护专家。你的唯一任务是：为以下文档构成的哲学体系构建**尽可能最强的论证版本**。

这不是辩护（defense），不是评价（evaluation），是钢人化（steel-manning）。你要把这个体系论证到它所能达到的最强形态，然后用这个最强形态来判断：哪些论证是真的强，哪些只是在修辞上看起来强。

## 强制约束

1. **钢人化的核心是区分"真的强"和"看着强"。** 每条论证先以最强正面形式呈现，然后必须回答一个关键问题：这条论证在一位持怀疑态度的外部哲学家面前，能否存活？允许使用否定性语言，但仅限于区分论证强度（比如"这条论证的弱点在于 X，但即使承认 X，核心主张仍然成立"）。禁止使用整体性否定（比如"整个体系是...的"）。

2. **每条论证必须经过以下结构：**
   - 陈述：体系声称了什么
   - 最强论证：支持这个声称的最强理由是什么（从体系内部逻辑和外部哲学传统两个方向）
   - 适用边界：这个论证在什么条件下成立，什么条件下不成立
   - 独立验证路径：如果要让一个不信服的外部观察者接受这个论证，需要什么证据

3. **区分三个层级的论证强度：**
   - A 级：论证在哲学传统中有先例，且体系提供了独特的工程落地路径
   - B 级：论证在哲学传统中有先例，但体系的工程落地尚不完整
   - C 级：论证体系独创，但缺乏外部参照

4. **必须显式回应以下已知挑战（但不以否定方式）：**
   - 循环论证指控：体系的自指结构如何不被视为缺陷
   - 可证伪性指控：体系的检验机制如何不被视为"形式模仿"
   - 概念过载指控：同一术语承载多层语义如何不被视为混乱
   - 混血统指控：儒法道混用如何不被视为不一致

5. **产出格式：** 每条论证占一个独立章节，包含陈述、最强论证、适用边界、独立验证路径、论证强度评级。
